API console

A small console for the live API. Create accounts, post a balanced transaction by picking accounts from a dropdown, and read balances and statements. Everything here calls the same public API at /v1.

Public demo: the ledger resets every 4 hours to fresh, realistic sample data. Anything you create is cleared on the next reset.
